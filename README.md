# GitHub Pages Profile Site

This repository contains a simple profile page structure for deploying on GitHub Pages.

## Structure

- `index.html` — home page
- `comp-card.html` — compact profile card page
- `projects.html` — featured projects page
- `assets/css/styles.css` — site styles
- `assets/js/main.js` — interactive scripts
- `assets/images/` — place your profile images and icons here

## Deploy

1. If this repo is named `your-username.github.io`, GitHub Pages will publish from the root.
2. Otherwise, enable Pages from the `main` branch and use the root or `docs/` folder.

## Customize

- Replace `Kira Malli` and placeholder text with your personal content.
- Add images to `assets/images`.
- Add more pages or sections as needed.
